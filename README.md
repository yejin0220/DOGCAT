# DOGCAT 프로젝트
![logo](https://github.com/yejin0220/DOGCAT/assets/117423376/f6f9a301-006f-47cd-a98a-5e93ea2e7cc4)

### 반려동물을 키우는 반려인들간의 소통을 통해 다양한 정보를 주고받고, 거래 및 유대감을 형성할 수 있는 커뮤니케이션 사이트

---

### 목차
* 기획의도
* 유사사이트 분석
* 유스케이스 다이어그램
* ERD
* 개발환경
* 주요기능
